PRISM Household Malaria Viewer

Interactive visualization of household-level malaria surveillance data from the PRISM (Program for Resistance, Immunology, Surveillance, and Modeling of Malaria) cohort study in Uganda (2011-2017).

Each visualization shows longitudinal malaria test results for all members of multi-person households, with one row per individual. Use the dropdown menu to navigate between households, which are sorted by total number of infection events.
